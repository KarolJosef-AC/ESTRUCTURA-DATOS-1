# Unidad 0: Estándares y Buenas Prácticas de Codificación

## 🎯 Objetivo
Aplicar estándares de codificación en Python para escribir código limpio, mantenible y profesional.

## 1. Estructuras de Codificación
### 1.1 Convenciones de nombres (PEP 8)
| Elemento  | Convención | ✅Correcto               | ❌ Incorrecto        |
| :-------- | :--------- | :------                  | :-------------        |
| Variables | snake_case | `resultado_final`        | `ResultadoFinal`      |
| Funciones | snake_case | `calcular_factorial()`   | `CalcularFactorial()` |
| Clases    | PascalCase | `ConjuntoEstatico`       | `conjunto_estatico`   |
